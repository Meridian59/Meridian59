Recommendations:

Note: these are based on the philosophy of

  • balancing the number of spells in each level across schools
  • balancing within each school the progression of the number of spells in each level
  • increasing the number of low-level spells
  • balancing the aggregate usage of each school
  • balancing within each school the progression of usage from level to level

Drawbacks of the data interpretation used to make these recommendations:

  • 'number of casts' doesn't necessarily reflect the usefulness of a spell. As such, this is only a valid heuristic for large differences, which unfortunately we have here.
  • The data crunched has been gathering for some time, so the new Renaissance spells haven't yet had a chance to make much of an impact. Further balancing in terms of spell/skill power (but not which ones we add) may become necessary as new data is collected.
